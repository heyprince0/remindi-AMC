# Quick Start - Quotations Update

## 🚀 Get Running in 3 Steps

### Step 1: Update Database (5 minutes)

1. Open Supabase dashboard
2. Click **SQL Editor** → **New Query**
3. Copy entire contents of `/lib/db-migrations.sql`
4. Click **Run**
5. Done! ✅

**If you have existing quotations:**
- See `DATABASE_MIGRATION.md` for migration steps

### Step 2: Deploy Code (2 minutes)

```bash
git add .
git commit -m "feat: Rebuild quotation form and PDF for professional Indian contractor layout"
git push
```

### Step 3: Test It (10 minutes)

1. Create a new quotation
2. Fill in customer: Name, Address, District, State, Pin Code
3. Fill in letter: Subject and Body (use suggestion buttons!)
4. Add items and calculate
5. Download PDF - verify it looks professional!
6. Done! ✅

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| Form shows old fields | Database migration not done |
| PDF not downloading | Check browser console for errors |
| GST shows NaN | This is fixed - clear cache and reload |
| Missing theme color in PDF | Set theme color in company profile settings |
| Logo not showing in PDF | Check logo URL in company profile |
