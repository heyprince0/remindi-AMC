## 🆘 Troubleshooting

| Issue | Solution |
|-------|----------|
| Form shows old fields | Database migration not applied |
| PDF not generating | Check browser console, ensure company profile exists |
| GST shows NaN | This should be fixed now, clear cache and reload |
| PDF missing logo | Check logo URL in company profile, verify it's accessible |
| Wrong theme color | Ensure company profile has theme_color set (default: #185FA5) |
| Amount in words wrong | Check grand_total calculation, verify subtotal is correct |
